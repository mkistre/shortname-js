# [shortname-js](https://github.com/moser-jose/shortname-js)

`shortname-js` is a JavaScript function designed to shorten names in a simple and user-friendly way. The function allows users to quickly enter full names and receive abbreviated versions of those names in return. üòÖ

### yarn

`npm install @mosmmy/shortname-js`
